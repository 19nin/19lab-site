# 19lab.app

Official site for 19lab (https://19lab.app).

## Stack

- Astro 5
- Tailwind CSS 3
- Cloudflare Pages (deployment target)

## Develop

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Output goes to `dist/`.

## Deployment

Configured for Cloudflare Pages.

- **Build command:** `npm run build`
- **Output directory:** `dist`
